# Weather App

## Description
The Weather App is a React-based application that provides real-time weather information for multiple cities. Utilizing the OpenWeatherMap API and Ant Design components, the app offers a sleek and user-friendly interface for viewing current weather conditions, forecasts, and more. The design is both intuitive and responsive, ensuring a great user experience on desktop and mobile devices.

## Features
- **Real-Time Weather Data:** Fetch and display current weather conditions using the OpenWeatherMap API.
- **Multi-City Support:** View weather information for multiple cities simultaneously.
- **User-Friendly Interface:** Built with Ant Design components for a modern and clean look.
- **Responsive Design:** Optimized for both desktop and mobile viewing.
- **Dynamic Updates:** Provides real-time updates to ensure the latest weather information.

## Technologies
- **Frontend:** React.js
- **UI Components:** Ant Design
- **Weather API:** OpenWeatherMap API
